(CSS) safari에서 drop-shadow가 짤리는 현상

2022.10.10
1분

Table of contents

  • 현상
    • 해결 방법

      현상

      style로 drop-shadow를 주입해줬지만 safari에서는 shadow가 짤린다.

      filter = 'drop-shadow(16px 16px 16px gray)';

      해결 방법

      브라우저가 fileter가 변경될 수 있음을 css로 알려주면 해결 가능하다.

      will-change: filter;
