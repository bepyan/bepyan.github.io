```
Element implicitly has an 'any' type because type 'typeof globalThis' has no index signature.
```

위와 같은 에러가 발생하면서 글로벌 변수를 사용할 수 없는 경우가 있다.

```ts:index.d.ts
import { Mongoose } from 'mongoose';

declare global {
  namespace NodeJS {
    interface Global {
      mongoose: Mongoose;
    }
  }
}
```
