`input`이 포커스 될 시 스타일은 [`outline`](https://developer.mozilla.org/ko/docs/Web/CSS/outline) 속성이 활용된다.

```css
input:focus {
  outline: none;
}
```

<br />

`tailwind`

```html
<input className="focus:outline-none" />
```
